WebDev Learning

En WebDev Learning nos especializamos en hacer de usted una persona experta en el ámbito del desarrollo de aplicaciones web, poniendo nuestro enfoque en el lenguage JavaScript, el framework "Angular" y en la lbrería "React".

JavaScript

JavaScript es un lenguaje de programación interpretado, dialecto del estándar ECMAScript. Se define como orientado a objetos, ​ basado en prototipos, imperativo, débilmente tipado y dinámico.

Angular

Angular es un framework para aplicaciones web desarrollado en TypeScript, de código abierto, mantenido por Google, que se utiliza para crear y mantener aplicaciones web de una sola página.

React

React es una biblioteca Javascript de código abierto diseñada para crear interfaces de usuario con el objetivo de facilitar el desarrollo de aplicaciones en una sola página. Es mantenido por Facebook y la comunidad de software libre. En el proyecto hay más de mil desarrolladores libres.

WebDev Learning

WebDev Learning fue fundada a mediados del 2022 por el estudiante de desarrollo web Wesley Cascante. Wesley lleva mucho tiempo inmerso en el mundo del desarrollo, aprendiendo lo suficiente antes de lanzarse a una empresa como desarrollador para comenzar a aplicar sus habilidades y tomar experiencia. WebDev Learning es un proyecto que tiene en marcha, en donde se promete a futuro una plataforma para que muchas personas de todo el mundo tengan acceso a un curso de calidad en donde podrán aprender lo requerido para formarse como desarrollador de aplicaciones web, uno de los trabajos mejor pagados en el mundo!
